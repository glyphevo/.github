# GlyphEvo Labs Community Defaults

This special `.github` repository stores the organization profile and default community health files for repositories under `glyphevo`.

## What This Repository Provides

- Organization profile content shown on the GlyphEvo Labs GitHub overview page.
- Default contribution guide.
- Default security policy.
- Default support policy.
- Default issue templates.
- Default pull request template.

Project-specific repositories may override these defaults with their own files when needed.
