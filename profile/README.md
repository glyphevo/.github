# GlyphEvo Labs

**Open-source infrastructure for AI agents, code intelligence, and context engineering.**

GlyphEvo Labs builds developer-oriented tools that help AI agents understand codebases, organize context, and work safely across complex software systems.

## Focus Areas

- **Code intelligence**: turning repositories into structured, navigable context for humans and agents.
- **Agent memory**: capturing decisions, traces, and project knowledge across development workflows.
- **Context engineering**: designing reliable ways to retrieve, compress, route, and apply context.
- **Open interfaces**: CLI-first, API-friendly, and agent-native tooling.

## Principles

- Local-first where possible.
- Agent-native by design.
- Clear interfaces over hidden magic.
- Practical tools before grand frameworks.
- Open-source foundations with room for commercial-grade reliability.
